# Screenshots

Place dashboard screenshots here for use in the README and documentation.

## Suggested screenshots

| Filename | What to capture |
|---|---|
| `dashboard-overview.png` | Full dashboard — Charts tab with candlestick chart and portfolio chart side by side |
| `backtest-controls.png` | Left panel — Backtest tab showing Stop Loss / Take Profit / Max DD Kill / date range sliders |
| `compare-algos.png` | Compare Algos tab — all 5 algorithms in the comparison table |
| `algorithms-tab.png` | Algorithms tab — DQN and DDQN cards showing live performance metrics and architecture |
| `live-simulation.png` | Charts tab mid-simulation — trade markers on candles, portfolio equity updating in real time |

## How to add

1. Take a screenshot of the running dashboard at http://localhost:3000
2. Save as a PNG file in this directory
3. Reference it in `README.md` using a relative path:

```markdown
![Dashboard overview](docs/screenshots/dashboard-overview.png)
```
